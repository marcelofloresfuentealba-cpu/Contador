Contabilizador de Productos

Este proyecto es una interfaz web sencilla diseñada para visualizar un producto (Laptop Gamer AMD) y gestionar la cantidad seleccionada para calcular un total a pagar.

Características:

* Visualización de Producto: Muestra una tarjeta con la imagen del producto y su precio base.
* Gestión de Cantidad: Permite al usuario aumentar o disminuir la cantidad de unidades deseadas mediante botones de control (+ y -).
* Cálculo en Tiempo Real: Actualiza automáticamente el total a pagar según la cantidad seleccionada.

Tecnologías Utilizadas:

* HTML5: Estructura del contenido.
* CSS3: Estilos básicos (referenciados desde ./assets/css/style.css).
* JavaScript: Lógica de funcionamiento del contador (referenciado en script.js).


Estructura del Proyecto:

* index.html: Archivo principal con la estructura de la interfaz.
* assets/css/style.css: Archivos de estilos para el diseño.
* assets/img/: Carpeta que contiene las imágenes (incluyendo Laptop.webp).
* script.js: Archivo con la lógica para manejar los eventos de los botones y el cálculo del total.

Cómo Utilizar
1.- Abre el archivo index.html en tu navegador web.
2.- Utiliza los botones + y - para ajustar la cantidad de Laptop Gamer AMD.
3.- El monto en la sección "Total a pagar" se calculará automáticamente en función del precio base ($400.000).
